Pull Request Template

Jul 4, 2022

Here is the pull request template that we use during our application development(specific to github). It lives inside the .github directory of the project, the file should be named pull_request_template.md. And every time a PR is opened, this gets pre-filled.


### Issue link
[Link]()

### Summary
_(Summarize the issue in couple sentences so others could be on the same page during the review. **What** is being changed. **Why** are we doing this? **How** are we doing this?)_

### QA instructions
_(How this issue could be tested: test data, test env name etc.)_


### Deploy instructions
_(If applied: shall a rake task be running, any migrations to be applied?)_
